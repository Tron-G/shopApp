# shopapp

练习项目，对vue2的系统全面学习，课程来源于coderwhy老师

项目知识点：

1. 组件化开发思维，封装，接口隔离，项目结构组织
2. 面向数据对象开发
3. 使用webpack手动搭建一个简单的vue脚手架
4. 路由，多页面开发，传参，动态路由
5. axios
6. 移动端适配，better-scroll代替原生滚动
7. 事件总线
8. tab-control吸顶效果，sticky属性失效的替代方法
9. 用户浏览状态的缓存，keep-alive失效的解决方法
10. mixin
11. nextTick深入理解
12. vuex
13. toast组件开发及插件化
14. 图片懒加载
